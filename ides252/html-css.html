HTML & CSS

HTML - Hypertext Markup Language

Semantics

HTML is a markup language. It defines the meaning of the pieces of content on a page.

Tags you should know

I have listed a few key tags below, but a comprehensive list of tags can be found here.

Tag What it means Syntax
p paragraph <p>This is a paragraph</p>
h1 - h6 Highest to lowest levels of headers <h1>This is a first level header</h1>
a anchor tag - use for hyperlinks <a href="http://google.com">This is an anchor tag for links</a>
img Image <img src="img/cute-cat.jpg" alt="Cute cat"> Note that the image tag does not come in pairs with an opening and closing. It is only a single tag.
table, and all the tags that fall within a table: thead (header section), tbody (table body), tr (rows), td (columns), th (columns in the thead section) Table tags <table>
  <tr>
   <td></td>
   <td></td>
  </tr>
</table>
div Division. Usually a block-level element. <div> ... something goes here ... </div>
span An inline element, usually used to separate or style inline content differently from the content surrounding it. <p>This is an example of <span>how a span is used</span></p>
ul and ol, with li Unordered lists and ordered lists. Using ordered lists places a number or letter for each list element (you can control whether it's a number of letter). <ul>
  <li>First list item</li>
  <li>Second list item</li>
  <li>Third list item</li>
</ul>

CSS - Cascading Style Sheets

CSS is used to define how things look. More specifically, it defines how the pieces of content defined by HTML structure should look.

This page has a great, short introduction to CSS syntax. Please go read it. You need to know the key concepts of selectors, declarations, properties, values, and css comments.

Activities for class

First, create a page according to some detailed instructions. Then work in groups of 4 or 5 - one of your group will explain how they built the page and what each of the elements means. Then one person from your group will be chosen to explain how you built your page to the class as well.

Make a small website of three pages, all linked together with a common menu made from a list. Pretend it's for a small coffee shop or restaurant. Home - Menu - About

Make a long scrolling page with headers and images - preferably to tell some kind of story. Use anchor tags to move from one part of the page to another (link back to top and link to a section). If you don't use a story, also pretend that it's for a small restaurant or coffee shop.

Make a small indie band website with 3-4 pages.
